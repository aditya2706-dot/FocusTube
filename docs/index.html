Installation

Up in 60 seconds

No Chrome Web Store account required. Just download, unzip, and load.

Step 01
⬇

Download the zip

Click the button below and save focustube-v1.2.0.zip anywhere on your computer. Then unzip it — you'll get a folder called focustube-v1.2.0.

Step 02
🧩

Open Chrome Extensions

In Chrome, go to chrome://extensions in the address bar. Toggle on Developer mode (top-right corner).

Step 03
📂

Load the folder

Click Load unpacked and select the unzipped focustube-v1.2.0 folder. The extension appears instantly. Pin it to your toolbar.

Ready to install?

Free, open source, no sign-up. Works on any Chromium browser (Chrome, Brave, Edge, Arc).
